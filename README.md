# CHIP-8 Emulator

### A learning project
This is a project created by me to learn about emulation. \
This program was not created to be usable, rather, for me to learn about hardware emulation. \
That said, there is a release, albeit unfinished. \
I may choose to update it in the future, for better user experience. 

![snake](/images/snaketitle.png)


## Features
1. Emulator can play ch8 games
2. User can set scale and delay if they choose
3. Small file size and open source.

![RPS](/images/RPS.png)

## Shortcomings
1. Not all games work. This is a problem with the emulator, and I will look into it.
2. No sound present.
3. No SuperChip implementation

![rockto](/images/rockto.png)

## Goals for the future
1. Fix bugs so that all games work correctly on the emulator
2. Implement sound (simple mono tone)
3. The user can set color palettes. controls, and other information, potentially in a config file.

Be on the lookout for future projects similar to this one. I have a few interesting ideas.
